# Roo Code Configuration

This directory contains Roo Code rule files for the animal advocacy platform.

## Contents

- **`rules/`** — 18 rule files combining mode-specific rules (`rules-architect.md`,
  `rules-code.md`, `rules-debug.md`, `rules-interview.md`, `rules-review.md`),
  domain concerns (advocacy language, security, privacy, accessibility, emotional safety,
  cost optimization, testing), and process skill files (code review, git workflow,
  plan-first development, requirements interview, security audit, testing strategy).
  All rules are consolidated into a single flat directory.
