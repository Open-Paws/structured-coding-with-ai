# Augment Code Configuration

This directory contains Augment Code instruction files for the animal advocacy platform.

## Contents

- **`rules/`** — 14 instruction files covering the full rule set: a `main.md` entry point,
  domain concerns (advocacy language, security, privacy, accessibility, emotional safety,
  cost optimization, testing), and process skills (code review, git workflow, plan-first
  development, requirements interview, security audit, testing strategy).
