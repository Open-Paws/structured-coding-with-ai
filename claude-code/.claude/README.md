# Claude Code Configuration

This directory contains all Claude Code scoped rules and skill packages for the animal advocacy platform.

## Contents

- **`rules/`** — 7 scoped rule files activated by file-path glob patterns or task context.
  Covers testing, security, privacy, cost optimization, advocacy domain language,
  accessibility, and emotional safety.

- **`skills/`** — 6 process skill packages, each in its own subdirectory with a `SKILL.md`
  workflow file. Skills cover code review, git workflow, plan-first development,
  requirements interview, security audit, and testing strategy.
