# Windsurf Configuration

This directory contains Windsurf rule files for the animal advocacy platform.

## Contents

- **`rules/`** — 14 rule files using Windsurf's YAML frontmatter with 4 trigger types:
  `always_on`, `glob` (path-based activation), `model_decision` (agent-selected), and
  `manual`. Includes a `main.md` entry point, domain concerns (advocacy language,
  security, privacy, accessibility, emotional safety, cost optimization, testing), and
  process skills (code review, git workflow, plan-first development, requirements
  interview, security audit, testing strategy).
