# Kilo Code Configuration

This directory contains Kilo Code rule files and skill packages for the animal advocacy platform.

## Contents

- **`rules/`** — Mode-specific rule files (architect, ask, code, debug, orchestrator),
  domain concern files (advocacy language, security, privacy, accessibility, emotional
  safety, cost optimization, testing), and a `memory-bank/` subdirectory for persistent
  project context across sessions.

- **`skills/`** — 6 process skill packages, each in its own subdirectory with a `SKILL.md`
  workflow file. Skills cover code review, git workflow, plan-first development,
  requirements interview, security audit, and testing strategy.
