# Project Brief — Animal Advocacy Platform

Software for animal liberation and advocacy organizations. Built by and for the movement.

## Domain

Animal advocacy — investigation documentation, public campaigning, coalition coordination, legal defense, and sanctuary management. This is a **high-risk domain**: investigation data is subpoena-target evidence under ag-gag statutes, activists face criminal prosecution, and system compromise endangers human and animal lives.

## Core Mission

Build tools that protect the people protecting animals. Every technical decision serves operational security, coalition resilience, and movement longevity.

## What Makes This Different

- **Three adversaries** — state surveillance, industry infiltration, AI model bias
- **Adversarial legal discovery** — design for court subpoenas, not just hackers
- **Coalition complexity** — multiple organizations with different risk profiles sharing data safely
- **Traumatic content** — investigation footage and witness testimony require emotional safety by design
- **Nonprofit constraints** — every compute dollar has an alternative use in direct advocacy
- **Hostile infrastructure** — unreliable internet, device seizure risk, mesh networking needs

## Ubiquitous Language

The codebase uses movement terminology. NEVER introduce AI-invented synonyms. See `context.md` for bounded contexts and the advocacy-domain concern file for the full language dictionary.
