# Cursor Configuration

This directory contains Cursor rule files for the animal advocacy platform.

## Contents

- **`rules/`** — 13 `.mdc` rule files using Cursor's MDC format with YAML frontmatter
  for `description`, `globs`, and `alwaysApply` fields. Includes domain concerns
  (advocacy language, security, privacy, accessibility, emotional safety, cost optimization,
  testing) and process skills (code review, git workflow, plan-first development,
  requirements interview, security audit, testing strategy).
