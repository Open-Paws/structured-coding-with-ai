# No Animal Violence Integration

Enforces compassionate language standards across the codebase using the Open Paws no-animal-violence toolkit.

## Components

- **Pre-commit hook**: Blocks commits with speciesist language
- **ESLint plugin**: Inline JS/TS detection
- **Vale rules**: Markdown/prose checking
- **Semgrep rules**: Cross-language static analysis
- **VS Code extension**: Real-time inline detection
- **GitHub Action**: CI PR validation
- **Reviewdog**: Inline PR comments

## Sources

All rules sourced from Open-Paws GitHub organisation repositories.
